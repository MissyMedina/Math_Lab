∫ Calculus Workspace Features

∫ Your Workspace

Problem Input: Calculus problems (derivatives, limits, etc.)

Solution Area: Detailed mathematical reasoning

Tools: Clear workspace, Save work

Tips: State rules, show steps, simplify answers

Example Usage:

Problem: Find derivative of f(x) = 3x² + 2x - 5

Given: f(x) = 3x² + 2x - 5
Step 1: Apply power rule to each term
Step 2: d/dx[3x²] = 3 · 2x¹ = 6x
Step 3: d/dx[2x] = 2 · 1 = 2
Step 4: d/dx[-5] = 0
Answer: f'(x) = 6x + 2
